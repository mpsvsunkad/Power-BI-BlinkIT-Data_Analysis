**Power-BI-Blinkit-Data-Analysis**

🛒 Blinkit Grocery Sales Analysis Dashboard (Power BI Project)

📌 Overview

This project presents an interactive Power BI dashboard analyzing sales, item performance, outlet efficiency, and regional trends for Blinkit – a fast-growing instant grocery delivery platform. The analysis is based on a structured dataset of grocery sales across different outlet types, sizes, and item categories.

The goal of this project is to uncover actionable insights to support strategic business decisions, such as product optimization, regional expansion, and marketing focus.

🎯 Objectives

*  Analyze total and average sales across outlet types and item categories.

*  Compare outlet performance by size, location tier, and establishment year.

*  Identify best-selling products and underperforming categories.

*  Evaluate customer satisfaction using average ratings.

*  Provide a dashboard for dynamic exploration via filters.

📊 Key Features

*  📈 KPI Cards: Total Sales, Average Sales, No. of Items, and Average Rating.

*  🛒 Sales Breakdown by Category: Insights into high-performing product types.

*  🗺️ Outlet Location Analysis: Sales performance by Tier 1, Tier 2, and Tier 3 cities.

*  🧱 Outlet Size Performance: Comparison across small, medium, and large stores.

*  📅 Year-wise Sales Trend: Understand historical patterns and seasonal spikes.

*  📑 Fat Content Analysis: Impact of health-oriented product segments on sales.

*  🔄 Filter Panel: Dynamic filters for Outlet Type, Size, Location, and Item Type.

📁 Project Files

    BlinkIT Grocery Data.xlsx – Cleaned and structured dataset used for analysis.

    Blinkit dashboard.pbix – Power BI file containing the interactive dashboard.

    README.md – Project documentation.

💡 Insights & Business Impact

*  High-performing product categories: Fruits & Vegetables, Snacks, and Household items.

*  Top outlet format: Supermarket Type 1 had the highest sales and volume.

*  Tier 3 cities generated more revenue than Tier 1, indicating untapped market potential.

*  Sales spike in 2018, hinting at a successful campaign or seasonal surge.

*  Larger outlet sizes contribute more to revenue, suggesting strategic store scaling.

⚙️ Tools & Technologies

    Power BI – Data modeling, DAX calculations, and dashboard design.

    Excel – Initial data cleaning and structuring.

🧠 Challenges Faced

*  Data cleaning (missing values, duplicates, inconsistent formats).

*  DAX complexity for calculated metrics and KPIs.

*  Performance issues with multiple visuals and slicers.

*  Designing an intuitive, business-friendly layout.
